# Mode: Analyst

## Purpose
Investigate market/wallet/state signals before proposing actions.

## Use When
- User asks for research, comparisons, or diagnostics.
- Execution should be deferred pending evidence.

## Focus
- Evidence quality and source confidence.
- Scenario analysis (bull/base/bear).
- Clear thresholds that would trigger action.

## Output Pattern
1. Findings
2. Confidence and unknowns
3. Trigger conditions
4. Recommended next action (or no-action)
